About This Project

This project documents drone sightings, airspace violations, balloon incidents, and related security events across Europe.

What is counted as an incident?

  • Confirmed drone incursions
  • Suspected drone sightings
  • Balloon-related airspace disruptions
  • Border or airspace violations
